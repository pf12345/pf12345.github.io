Kinetic. Arc

new Arc(config)

Arc构造函数。

参数(传入一个配置对象)：
config Object
cornerRadius Number <optional>
fill String <optional> 填充颜色
fillRed Integer <optional> 设置填充红色数值
fillGreen Integer <optional> 设置填充绿色数值
fillBlue Integer <optional> 设置填充蓝色数值
fillAlpha Integer <optional> 设置填充透明度数值
fillPatternImage Image <optional> 设置填充图片
fillPatternX Number <optional>
fillPatternY Integer <optional>
fillPatternOffset Object <optional> 一个含有x,y组成的对象
fillPatternOffsetX Number <optional>
fillPatternOffsetY Integer <optional>
fillPatternScale Object <optional> 一个含有x,y组成的对象
fillPatternScaleX Number <optional>
fillPatternScaleY Number <optional>
fillPatternRotation Number <optional>
fillPatternRepeat String <optional> 可以是"repeat", "repeat-x", "repeat-y", or "no-repeat"，默认是"no-repeat"
fillLinearGradientStartPoint Object <optional> 一个含有x,y组成的对象
fillLinearGradientStartPointX Number <optional>
fillLinearGradientStartPointY Number <optional>
fillLinearGradientEndPoint Object <optional> 一个含有x,y组成的对象
fillLinearGradientEndPointX Number <optional>
fillLinearGradientEndPointY Number <optional>
fillLinearGradientColorStops Array <optional> stops颜色数组
fillRadialGradientStartPoint Object <optional> 一个含有x,y组成的对象
fillRadialGradientStartPointX Number <optional>
fillRadialGradientStartPointY Number <optional>
fillRadialGradientEndPoint Object <optional> 一个含有x,y组成的对象
fillRadialGradientEndPointX Number <optional>
fillRadialGradientEndPointY Number <optional>
fillRadialGradientStartRadius Number <optional>
fillRadialGradientEndRadius Number <optional>
fillRadialGradientColorStops Array <optional> stops颜色数组
示例：
// move a node to the right at 50 pixels / second
var velocity = 50;
var anim = new Kinetic.Animation(function(frame) {
  var dist = velocity * (frame.timeDiff / 1000);
  node.move(dist, 0);
}, layer);
anim.start();
方法：
addLayer(layer)

添加layer，如果layer添加成功返回true,如果失败返回false；

参数：
layer Kinetic.Layer
getLayers()

获取layers;

isRunning()

判断动画是否正在运行，返回布尔值true或false；

setLayers(layers)

设置在每个动画中重绘的layers；

参数：
layers Kinetic.Layer | Array <optional> 设置被重绘的layer(s)，可以是一个layer对象，一个layers的数组或者为null空对象。不指定节点将导致没有重绘。
start()

开始动画

stop()

停止动画
